# Embedded Systems & 8051 Microcontroller Programs

## 📌 Overview

This repository contains **Embedded C and Assembly programs** developed using the **8051 Microcontroller**.
It covers fundamental interfacing techniques, peripheral communication, and mini embedded system projects useful for **Electronics & Telecommunication Engineering students**, beginners in embedded systems, and academic laboratory work.

The programs demonstrate practical implementation of **timers, interrupts, displays, communication protocols, sensors, and real-time applications**.

---

## 🛠️ Technologies Used

* Embedded C
* Assembly Language (A51)
* 8051 Microcontroller
* Keil µVision IDE
* Proteus Simulation (optional)

## 📁 Important Files

* `STARTUP.A51` → 8051 startup assembly file
* `delay.c` → Delay generation functions
* `delay.uvproj / .uvopt / .uvgui` → Keil project configuration files
* `Listings / Objects` → Compiled output files

---

## ▶️ How to Run

1. Install **Keil µVision IDE**.
2. Open the `.uvproj` file.
3. Select target device (8051).
4. Build the project.
5. Simulate using Keil Debug Mode or Proteus.
